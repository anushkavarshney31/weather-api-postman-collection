Weather Map API

A Postman collection that tests the OpenWeatherMap API for London, UK.

Tests Included

1. Verify Latitude is 51.51 and Longitude is -0.13
2. Verify Status Code is 200
3. Verify City Name is London
4. Verify Weather Description Exists

How to Run

1. Import `WeatherMapAPI.postman_collection.json` into Postman
2. Click **Run collection** to execute all 4 tests
